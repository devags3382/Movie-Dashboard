## Project Overview

The entertainment industry generates massive amounts of data related to movie ratings, popularity, genres, languages, revenue, and audience preferences. Understanding this data can help production companies, streaming platforms, and distributors make informed decisions regarding content creation, marketing strategies, and audience targeting.

This project analyzes movie data and presents interactive visualizations to uncover trends in movie popularity, ratings, genres, languages, runtime, and revenue. The dashboard provides valuable insights into audience preferences and movie performance across different categories.

---

# Problem Statement

Explore different aspects of movie data such as popularity, ratings, genres, languages, runtime, release dates, and revenue. Create interactive reports and visualizations to identify trends and patterns that influence movie success.

This project helps develop skills in data analysis, data visualization, and business intelligence while providing insights that can support data-driven decision-making in the entertainment industry.

---

# Objectives

* Analyze movie distribution across genres and languages.
* Identify top-rated movies based on ratings and vote counts.
* Study movie popularity trends over time.
* Evaluate revenue performance across genres.
* Understand audience preferences based on ratings and popularity.
* Analyze movie runtime trends by release year.
* Compare language distribution and audience engagement.
* Provide actionable recommendations for content selection and scheduling.

# Dataset Description

| Column            | Description                                                     |
| ----------------- | --------------------------------------------------------------- |
| Movie ID          | Unique identifier for each movie                                |
| Movie Title       | Name of the movie                                               |
| Original Language | Language in which the movie was originally produced             |
| Genre             | Movie category such as Action, Drama, Comedy, etc.              |
| Popularity        | Popularity score based on audience engagement and user activity |
| Release Date      | Date the movie was released                                     |
| Rating            | Average rating assigned by viewers or critics                   |
| Vote Count        | Number of votes received                                        |
| Runtime           | Duration of the movie in minutes                                |
| Revenue           | Revenue generated from box office and other sources             |

# Key Insights

## Language Popularity

* Movies produced in **English received the highest number of audience votes**.
* English-language movies dominate audience engagement and participation.

---

## Highest Rated Movies

* The two highest-rated movies in the dataset are produced in **Arabic** and **Telugu**.
* These movies achieved exceptional audience ratings despite lower representation compared to English-language films.

---

## Runtime Analysis

* The average runtime of movies is approximately **99.88 minutes**.
* Most successful movies fall within a runtime range that audiences find comfortable and engaging.

---

## Genre Preferences

The three most popular genres among audiences are:

1. Drama
2. Action
3. Comedy

These genres consistently attract higher viewership and audience engagement.

---

## Popularity Analysis

* Korean movies have the highest popularity scores in the dataset.
* This indicates strong audience interest and engagement with Korean cinema.
